# jin RPG 구상자료

이 폴더는 `진(jin) RPG`의 기획 내용을 계속 누적하는 작업 공간이다.

## 파일 관리 규칙

- 기존 최신 파일을 기준으로 다음 버전을 만든다.
- 파일명은 `YYYY-MM-DD_vN_내용.md` 형식을 사용한다.
- 엑셀 기획표는 필요할 때 유지하되, 이후 진행 내용은 마크다운 문서에 먼저 누적한다.
- 구현 중 결정된 내용만 적는다.
- 쓰지 않기로 한 기능은 별도 제외 목록으로 길게 관리하지 않는다.
- 오래된 파일을 덮어쓰지 않고 새 버전 파일을 추가한다.

## 현재 버전 기준

- v2: `coremaster_rpg_design_plan_v2.xlsx`
- v3: `2026-06-11_v3_jinrpg_구상.md`

## 작업 방식

1. 최신 버전 파일을 먼저 읽는다.
2. 새 아이디어나 결정이 생기면 다음 버전 파일을 만든다.
3. 새 버전에는 이전 핵심 내용을 유지하고, 변경/추가 내용을 반영한다.
4. 실제 구현 전에는 최신 구상 파일 기준으로 MVP 범위와 DB 변경 여부를 확인한다.
